# Online-Movie-Ticket-Booking-System
SQL-based Online Movie Ticket Booking System with sample data and queries
This is a **SQL project** for managing movies, theaters, shows, and bookings.
## Features
- Add movies, theaters, and showtimes
- Book tickets for users
- Check available seats
- Generate reports: top-booked movies, revenue, bookings per user

## Files
- `MovieBookingDB.sql` → Contains database, tables, and sample data
- `SampleQueries.sql` → Contains queries for reports and testing

## How to Run
1. Open MySQL Workbench or phpMyAdmin.
2. Run `MovieBookingDB.sql` to create the database and insert sample data.
3. Run `SampleQueries.sql` to execute queries and see outputs.
